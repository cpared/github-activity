# Github Activity

### Summary
This is a CLI application that fetchs data from a github API with a given user and displays into the terminal.

### How to run?
First of all, you have to clone the repository

```bash
git clone git@github.com:cpared/github-activity.git
```

Then you have to run this CLI with the following command in a terminal

```bash
go run ./cmd/cli <username>
```

Where `<username>` is the name of an specific github user.
